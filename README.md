### `Generative AI · Agentic Systems · LLM Engineering`

Building production-grade LLM systems and agentic architectures.
I work end-to-end across the stack — Python on the backend for agents,
RAG, and evaluation pipelines; Next.js on the web; SwiftUI when the
product needs to live on iOS. My focus is on the parts that actually
break in production: tool orchestration, long-horizon memory,
evaluation, and tracing.
